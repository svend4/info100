## 1. ЮРИДИЧЕСКИЙ ИИ-АССИСТЕНТ ДЛЯ ИНВАЛИДОВ В ГЕРМАНИИ

**Источник:** ~300 разговоров (серия «111 социал»), 7.4 МБ текста
**Уровень готовности:** 5/5 — готовые документы, шаблоны, судебная практика

**Что уже создано:**
- **Мегашаблон карточки услуги на 24 пункта** — готовая структура для любой социальной услуги с правовой базой (SGB IX, XII, V, VIII, UN-BRK)
- **Каталог 70+ модулей социальных услуг** с перекрёстным маппингом на 10-пунктовую классификацию
- **50+ готовых шаблонов** на немецком: Widerspruch, Stellungnahme, Klage, Antrag auf Persönliches Budget, Fristverlängerung, Nachzahlung
- **Master Dossier v1.0** — заполняемый шаблон личного дела по соцсистеме
- **Каталог 50+ видов ассистентских услуг** по SGB IX/XI/XII с расчётами стоимости
- **Анализ судебной практики BSG** по персональному бюджету
- **Анализ Kostenschieberei** — незаконное перекладывание расходов между ведомствами
- **Полный каталог соцуслуг РФ** (66 позиций, 8 видов) для сравнения

**Почему перспективно:**
- 13 млн людей с инвалидностью (GdB) в Германии
- Pflegeversicherung тратит €50+ млрд/год
- Аналогов (правовой ИИ + шаблоны + калькулятор) на рынке почти нет
- Готовый контент для MVP за 2-3 месяца

**Форматы монетизации:**
- SaaS для соцработников и юристов (€19-49/мес.)
- Telegram-бот / приложение для инвалидов (freemium)
- Консалтинг и обучающие курсы
- White-label для благотворительных организаций

## 3. РОЙ ДОМАШНИХ РОБОТОВ — «ЗООПАРК» ИЗ 15 СПЕЦИАЛИЗИРОВАННЫХ РОБОТОВ

**Источник:** разговор 708 и смежные (~10 разговоров)
**Уровень готовности:** 4-5/5 — YAML-манифесты, архитектура оркестратора

**Что уже создано:**
- **15 уникальных роботов** с именами, функциями, спецификациями:
  - Снегирь (доставка мелких предметов), Бобр (уборка), Барсук (сортировка), Стриж (скоростной курьер) и т.д.
- **YAML-манифест каждого робота:** масса, IP-рейтинг, навыки, сенсоры, энергетика
- **Архитектура оркестратора:** ROS2/DDS + MQTT + Matter, Contract Net Protocol
- **Стандарты:** зарядка, безопасность, коммуникация между роботами
- **Экономическая модель** поэтапного внедрения

**Почему перспективно:**
- Альтернатива дорогим гуманоидным роботам (каждый «зверь» — €50-500 вместо €50 000)
- Модульность: покупаешь по одному, наращиваешь «стаю»
- Патентабельная концепция
- Рынок домашних роботов: $12 млрд к 2030

**Источник:** разговор 5 (1766 сообщений, 4.1 МБ) + ~40 смежных
**Уровень готовности:** 4/5 — полное ТЗ, MVP Backlog

**Что уже создано:**

- **Аналогия «паспорт блока»:** каждое приложение как атомарная функция со входами/выходами/зависимостями/рисками
- **Реестр блоков** (CSV, 60 записей для WordPress/Make/n8n)
- **10 готовых Blueprint-связок** «one-click»: лиды, заказы, бэкапы, API
- **MVP Backlog** (CSV с эпиками и задачами)
- **Четырёхуровневая пирамида знаний** (Стратегия → Тактика → Оперативный → Практический)

**Почему перспективно:**
- Решает реальную проблему: 50 000+ WordPress-плагинов, 400+ коннекторов Make.com — но нет единого реестра с описанием рисков и совместимости
- Сравнение с Oslo Manual (OECD) — «рекомбинантная инновация»
- Реализуемо как SaaS за €100-150K
