## ВОЛНА 1: БЫСТРЫЕ ДЕНЬГИ (0-6 мес.)

### 1.1 Электронная книга «Persönliches Budget: Полный путеводитель»
- **Что делать:** Собрать из 300 разговоров структурированное руководство (200-300 страниц)
- **Языки:** Русский + Немецкий
- **Платформа:** Gumroad, Amazon KDP
- **Цена:** €29-49
- **Прогноз:** 50-200 продаж/мес. = €1 500-10 000/мес.
- **Затраты:** €0 (контент есть, вёрстка — ИИ)
- **Срок:** 2-4 недели

### 1.2 Telegram-бот «SozialHelper»
- **Что делать:** RAG-бот на базе накопленных шаблонов и каталогов
- **Стек:** Python + Aiogram + Ollama/Claude API + ChromaDB
- **Функции:** поиск по SGB, генерация Widerspruch, калькулятор бюджета
- **Цена:** Бесплатно (базовый) + €4.90/мес. (генерация документов)
- **Прогноз:** 500+ пользователей за 3 мес.
- **Затраты:** €200-500 (API)
- **Срок:** 4-6 недель

### 1.3 Консалтинг на Upwork/Fiverr
- **Что делать:** Услуга «Написание Widerspruch / Antrag по SGB IX»
- **Цена:** €50-150 за документ
- **Прогноз:** 5-10 заказов/мес. = €250-1500/мес.
- **Затраты:** €0
- **Срок:** Сразу

## ВОЛНА 2: ПРОДУКТ (6-18 мес.)

### 2.1 SaaS «SozialPlaner» для соцработников
- **Что делать:** Веб-платформа с каталогом услуг, генератором документов, таймлайном дела
- **Стек:** Next.js + Supabase + Claude API
- **Цена:** €29-99/мес. за рабочее место
- **Первые клиенты:** 3 Pflegedienste из Дрездена (личные контакты)
- **Финансирование:** EXIST-Gründerstipendium (€150K/год)
- **Затраты:** €20-40K (MVP)
- **Срок:** 4-5 месяцев

### 2.2 SmartOfficeProto (Виртуальный офис)
- **Что делать:** PWA с трёхпанельным интерфейсом (рабочая зона + ИИ + Kanban)
- **Стек:** React + Tauri + Ollama
- **Цена:** €9-19/мес.
- **Первые клиенты:** Фрилансеры из Reddit, Product Hunt
- **Затраты:** €3-5K (минимальный MVP)
- **Срок:** 3-4 месяца

### 2.3 Маркетплейс n8n-сценариев
- **Что делать:** Платформа продажи готовых автоматизаций
- **Стек:** Next.js + Stripe
- **Цена:** 20% комиссия, сценарии €5-50
- **Затраты:** €5-10K
- **Срок:** 3 месяца

## ВОЛНА 3: МАСШТАБ (18-36 мес.)

### 3.1 SkyMediaHub Bavaria (стартап)
- **Что делать:** Подать в Bayern Innovativ, Horizon Europe, EIC Accelerator
- **Первый продукт:** TetraDrone образовательный кит (5 дронов + ПО)
- **Цена кита:** €500-2000
- **Финансирование:** Грант €500K-2.5M
- **Партнёры:** Fraunhofer, TechHUB SVI (уже есть письмо!)

### 3.2 CareMate Total (B2B SaaS + IoT)
- **Что делать:** Пилот с 3 домами престарелых
- **Финансирование:** AAL Programme (EU), DiGA-сертификация
- **Цена:** €4.90/койка/мес.
- **Рынок:** 15 000+ учреждений в Германии

### 3.3 Рой роботов-зверей (hardware startup)
- **Что делать:** Прототип 2-3 ключевых роботов (Снегирь, Бобр)
- **Финансирование:** Kickstarter + EIC Accelerator
- **Цена:** €99-499 за робота
- **Срок:** 24-36 месяцев до первых продаж

## ИТОГОВАЯ ОЦЕНКА

**Общий объём проработанных идей:** экстраординарный.
**Главная проблема:** слишком много направлений, недостаточно фокуса.
**Главная рекомендация:** **фокус на одном продукте** (SozialPlaner или книга по SGB) → первые деньги → реинвестиция в следующие проекты.

Автор обладает **редким сочетанием компетенций** (право + технология + личный опыт), которое делает его потенциально единственным человеком в Германии, способным создать качественный ИИ-продукт для людей с инвалидностью. Это конкурентное преимущество, которое нужно монетизировать **сейчас**, пока рынок не заполнился.
