# Архив знаний — 1105 разговоров

Этот репозиторий содержит извлечённый текст из двух экспортов ChatGPT и аналитику по ним.

**Предыстория:** Это третья попытка сохранить результаты многолетней интеллектуальной работы. Два предыдущих архива (OSINT, изобретения, научные находки) были потеряны — один при госпитализации (~2010), второй при эмиграции/войне (~2022). Подробнее: `MANIFEST.md`
