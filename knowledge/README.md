# 📚 Knowledge — База Знаний

## Структура

- `raw/` — Сырые данные из внешних источников (клоны репозиториев)
- `processed/` — Структурированная информация (очищенные данные)
- `methods/` — Абстрактные методологии (паттерны, подходы)
- `graph/` — Граф связей между концепциями (JSON/GraphML)

## Поток данных

```
raw/ → processed/ → methods/ → graph/
```

## Граф знаний

Файл `graph/concepts.json` содержит:
- `nodes` — концепции (методологии, проекты, идеи)
- `edges` — связи между ними (использует, реализует, относится к)

Пример связи:
```json
{
  "from": "tetradron",
  "to": "parallel-bidirectional",
  "relation": "uses",
  "strength": 0.7
}
```
