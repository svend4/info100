# info100 — Эволюционная Лаборатория Агентов

> *«Мы не просто пишем код. Мы выращиваем мыслящие системы.»*

## 🧬 Что это?

**info100** — это монорепозиторий для тестирования, эволюции и валидации агентов ИИ. Здесь:

- 🤖 **Агенты эволюционируют** — от базовых к более сложным через скрещивание и мутации
- 🔄 **Внешние репозитории** превращаются в структурированные методологии  
- 🧪 **Эксперименты** автоматически документируются и воспроизводятся
- 📊 **Метрики** определяют, какие агенты выживают, а какие уходят в архив

## 🗺️ Структура

```
info100/
├── 🧠 agents/          # Экосистема агентов (core → evolved → variants)
├── 🔄 pipelines/       # Потоки обработки: ingest → distill → synthesize → validate
├── 📚 knowledge/       # База знаний: граф связей между концепциями
├── 🛠️ skills/          # Генерируемые артефакты (скиллы, плагины, модули)
├── 🧪 experiments/     # Датированные эксперименты с полной историей
├── 🎯 benchmarks/      # Тесты эффективности агентов
└── 📝 meta/            # Мета-информация: лог эволюции, метрики, решения
```

## 🚀 Как это работает

### 1. Ingest (Забор)
Агент берёт внешний репозиторий (например, `svend4/info1`), извлекает паттерны и записывает в `knowledge/raw/`

### 2. Distill (Перегонка)
Из сырых данных извлекается абстрактная методология → `knowledge/methods/`

### 3. Synthesize (Синтез)
Методологии скрещиваются: `TetraDrone × Parallel-Bidirectional = Swarm-Coder-v3`

### 4. Validate (Валидация)
Новый скилл тестируется → если работает, идёт в `skills/tested/`, если нет — в `retired/` с причиной

## 🎯 Принципы

| Принцип | Описание |
|---------|----------|
| **Эволюция, не революция** | Каждая версия агента — потомок предыдущей с мутациями |
| **Всё тестируется** | Нет скиллов без бенчмарков. Нет агентов без fitness_score |
| **Прозрачность** | Каждое решение агента можно отследить до источника |
| **Репродуцируемость** | Любой эксперимент можно повторить по дате и коммиту |

## 🏁 Быстрый старт

```bash
# Запустить сборщика для внешнего репо
python pipelines/ingest/info1-collector.py

# Сгенерировать новый скилл из методологии
python pipelines/synthesize/skill-generator.py --method parallel-bidirectional

# Запустить бенчмарк
python benchmarks/run.py --skill swarm-coder-v3
```

## 🧬 Генетический код

Агенты здесь — не просто код. Это «геномы»:

```yaml
# agents/evolved/alpha-7.yaml
genome:
  parent: [researcher-v2, critic-v1]
  mutations:
    - added: "self-reflection-loop"
    - removed: "verbose-output"
  fitness_score: 0.87
  lineage: [base, v1, v2, alpha-7]
```
